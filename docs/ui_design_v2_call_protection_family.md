# SoriCall 화면설계 v2 — 보이스피싱 통화 보호 가족

## 1. 핵심 용어

- **보이스피싱으로부터 보호할 가족**: 의심전화 탐지·경고·차단 서비스를 실제로 사용하는 부모 또는 조부모
- **의심전화를 확인해 줄 가족**: 보호할 가족에게 의심전화가 왔을 때 실제 통화 여부를 확인해 주는 가족
- **통화 보호 휴대전화**: 보호할 가족이 사용하며 SoriCall 통화 보호가 활성화된 Android 단말

화면에서는 `어르신`, `피보호자`, `Guardian` 같은 시스템 용어를 사용하지 않는다.

## 2. 최소 가족관계

### 보이스피싱으로부터 보호할 가족

`아버지 / 어머니 / 할아버지 / 할머니 / 기타`

### 의심전화를 확인해 줄 가족

관계의 기준은 항상 보호할 가족이다.

`아들 / 딸 / 손자 / 손녀 / 배우자 / 기타 가족`

## 3. 전체 화면 흐름

```text
시작 → 회원가입 → 필수동의 → 로그인
→ 보이스피싱으로부터 보호할 가족 등록
→ 의심전화를 확인해 줄 가족 등록
→ 가족 음성 등록 → 얼굴 등록(선택) → 통화 보호 홈
→ 정상전화 / 의심전화 분석 / 고위험 차단
→ 가족 확인 요청·응답 → 통화기록
```

역할 선택 화면과 senior_id 직접 입력 화면은 사용하지 않는다. 로그인 계정과 가족 등록정보로 역할과 단말을 자동 연결한다.

## 4. 화면별 설계

### 시작·회원가입·동의·로그인

- 시작: `보이스피싱 가족 사칭 전화를 함께 확인합니다.`
- 회원가입: 이름, 휴대전화 또는 이메일, 비밀번호, 본인확인
- 필수동의: 이용약관, 개인정보, 가족 전화번호, 음성 특징정보, 통화 위험분석
- 선택동의: 얼굴정보, 마케팅
- 로그인 후 등록 상태에 따라 가족 등록 또는 홈으로 자동 이동

### 보이스피싱으로부터 보호할 가족 등록

```text
보이스피싱으로부터 누구의 전화를 보호할까요?
[아버지] [어머니] [할아버지] [할머니] [기타]

성함 [          ]
휴대전화 [          ] [인증]
[다음]
```

정보 등록과 단말 보호 활성화를 구분한다. 단말 로그인이 끝나기 전에는 `통화 보호 준비 중`으로 표시한다.

### 의심전화를 확인해 줄 가족 등록

```text
김영희 어머니에게 의심전화가 오면 누가 확인할까요?
[아들] [딸] [손자] [손녀] [배우자] [기타]

성함 [          ]
휴대전화 [          ] [인증]
확인 순서 [가장 먼저]
[가족 등록]
```

### 음성·얼굴 등록

- 음성은 가족별 안내 문장 녹음, 품질 확인, 재녹음 제공
- 얼굴은 명확한 선택 기능으로 분리하고 `나중에` 제공
- 등록 원본 보존 여부와 삭제 정책을 화면에 표시

### 통화 보호 홈

```text
김영희 어머니의 전화를
보이스피싱으로부터 보호하고 있습니다.

통화 보호 켜짐
확인 가족 2명
가족 음성 등록 완료
```

### 정상전화

`등록된 가족 전화입니다. 안전하게 통화하세요.`

### 의심전화 분석

`가족 사칭이 의심됩니다. 돈을 보내거나 앱을 설치하지 마세요.`

번호, 음성, 합성음, 위험표현, 가족 확인 상태를 쉬운 문장으로 단계 표시한다.

### 고위험 차단

`보이스피싱 위험이 높아 전화를 차단했습니다.`

주요 행동은 `저장된 가족에게 다시 전화`와 `확인 가족에게 요청` 두 개로 제한한다.

### 가족 확인 요청

`지금 김영희 어머니께 전화하셨나요?`

- 네, 제가 전화했어요 (`CALLED`)
- 아니요, 제가 아닙니다 (`NOT_CALLED`)
- 잘 모르겠습니다 (`UNKNOWN`)

### 통화기록

정상·주의·위험·차단, 번호 끝 4자리, 통화 시각, 확인 응답, 대응 결과를 제공한다. 전사문은 기본 숨김 및 TTL 삭제한다.

### 관리자 Web

통화 보호 사용자, 확인 가족, 보호 단말 상태, 위험 통화, FCM 실패, AI 모델, 정책 버전, 감사로그, 개인정보 파기 현황을 관리한다.

## 5. 플랫폼 역할

- Web: 가입, 가족 등록, 음성·얼굴 등록 관리, 기록, 관리자 기능
- Android: 보호 단말 자동등록, 수신전화 경고·차단, 재통화, 가족 확인 푸시 응답
- 공통: 동일 용어·색상·위험등급·API 상태코드 사용
